Expand description

Internal trait for drivers to write into frames.

This trait is not intended for public use - clients should only use the read-only LidarFrame trait methods.

fn buffers_mut( &mut self, ) -> (&mut [f32], &mut [f32], &mut [f32], &mut [u8], &mut [f32])

Get mutable access to all buffers at once for SIMD writes.

Returns (x, y, z, intensity, range) slices. This method allows borrowing all buffers simultaneously, which is necessary for efficient SIMD processing where we need to write to multiple arrays in a single pass.
